``dpmcore.services.layout_exporter``
====================================

Export annotated DPM table layouts to Excel workbooks (hierarchical
headers, data-point cells, dimensional annotations, and categorisation
tooltips).

.. note::

   Requires the ``export`` extra: ``pip install dpmcore[export]``.
